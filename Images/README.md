# STM32F446RE Custom Development Board

## Overview

A custom STM32F446RE ARM Cortex-M4 development board designed using Altium Designer for embedded systems, robotics, industrial automation, and CAN communication applications.

## Features

- STM32F446RE Microcontroller
- ARM Cortex-M4 Core
- CAN Bus Support
- USB Interface
- SWD Debugging
- Custom PCB Design
- Manufacturing Ready Gerber Files
